# Weather Forecast App

A **weather forecast web application** that displays real-time weather data and a 5-day forecast using the SheCodes Weather API. This project is the **final project** for the SheCodes Intermediate course.

![Weather App Screenshot](Assets/Screenshots/screenshot.png)

## Features

- Search for any city worldwide and get its current weather.
- Displays:
  - City name
  - Current temperature
  - Weather condition (e.g., clear sky, rain, etc.)
  - Time
  - Humidity
  - Wind speed
  - Weather icon
- Shows a **5-day weather forecast**, including today and the next 4 days.

## Technologies Used

- HTML
- CSS
- Vanilla JavaScript
- SheCodes Weather API

## How to Use

1. Open the app on [Netlify](https://best-meteo-app.netlify.app)
2. Type the name of any existing city in the search bar (uppercase or lowercase works).
3. Hit **Search** to see the current weather and the 5-day forecast.

> Optionally, you can also clone this repository and open `index.html` in your browser to run it locally.
